# Smart-Garbage-Bin-Prediction
It is an ML binary classification system wherein I used Random Forest and XGBoost to predict the fill levels of smart city waste bins.
It is a Machine Learning binary classification system built to optimize the routes of municipal solid waste collection vehicles in smart cities.
I trained and evaluated both Random Forest and XGBoost architectures and I found XGBoost to be the optimal production model.
Random Forest: Accuracy= 86.67% ; Precision= 83.82% ; Recall= 66.28% ; F1 Score= 0.7403
XGBoost Classifier : Accuracy= 90.00% ; Precision= 86.84%% ; Recall= 76.74% ; F1 Score= 0.8148
Tech Stack and Libraries used:-
Platform : Google Colab
Language : Python 3
Libraries : 'xgboost' , 'scikit-learn', 'pandas' , 'numpy' , 'seaborn' , 'matplotlib'
